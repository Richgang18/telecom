## 📱 MOBILE SETUP (5 STEPS)

### 1. Download
- **Android:** Google Play Store → "Linphone"
- **iOS:** App Store → "Linphone"
- Install and open

### 2. Add Account
- Tap **☰ Menu** → **Settings** → **Accounts**
- Tap **+ Add Account**
- Select **"Use SIP Account"**

### 3. Enter Details
```
Username:     [Your extension: 101 or 102]
Password:     [Your password from above]
Domain:       pbx.vouchersdept.com
```

### 4. Advanced Settings
- Tap **"Advanced"** or **"Transport Settings"**
```
Transport:    TLS
Port:         5061
```

### 5. Enable Permissions
- **Android:** Settings → Apps → Linphone
  - ✓ Microphone
  - ✓ Phone
  - ✓ Notifications
  - ✓ Disable battery optimization

- **iOS:** Settings → Linphone
  - ✓ Microphone
  - ✓ Notifications
  - ✓ Background App Refresh

- ✅ Done!

---

## 🔧 QUICK TROUBLESHOOTING

### Not Registered (Red Status)?
1. Check internet connection
2. Verify username/password
3. Check Transport = **TLS**, Port = **5061**
4. Restart Linphone

### No Audio?
1. Check headset is connected
2. Test audio in Settings → Audio
3. Check volume levels
4. Try different headset

### Calls Not Coming Through?
1. Check status is GREEN
2. Check notifications enabled
3. Keep app running in background
4. Disable battery optimization (Android)
