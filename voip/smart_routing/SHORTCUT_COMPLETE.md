## 🎨 Customization Options

### Change Icon:
1. Replace `app_icon.ico` with your own
2. Right-click shortcut → Properties → Change Icon
3. Browse to new icon

### Change Name:
1. Right-click shortcut
2. Select Rename
3. Type new name

### Pin to Taskbar:
1. Launch app via shortcut
2. Right-click taskbar icon
3. Select "Pin to taskbar"

### Add to Start Menu:
1. Press Win+R
2. Type: `shell:programs`
3. Copy shortcut there
